原来：
Docker镜像的缺省命令是bash，如果不加 -it，bash命令执行了自动会退出，加-it后docker命令会为容器分配一个伪终端，并接管其stdin/stdout支持交互操作，这时候bash命令不会自动退出
像不使用docker-compose，我们会执行类似如下的命令
docker run -d -it mmm /bin/bash
