1 使用背景

在使用git的时候，经常会用到一个命令，那就是git commit -m 'msg'，但是这个msg还是有要求的，那就是在push代码的时候，msg不能少于5个字符（中英文都算一个字符），但是在执行git commit的时候，这个msg是可以写任意个字符的，也就是说当你commit的时候，没有写够5个字符，那么在push的时候，就会提示不能push，所以就需要用到修改已提交 commit 的信息。
