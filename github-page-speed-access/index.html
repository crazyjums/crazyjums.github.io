如何加速访问GitHub page页面
Created 2020-01-01| Updated 2021-09-06
Word count:140|Reading time: 1 min
|Post View:

1 查询地址

打开https://www.ipaddress.com/ 查询以下三个链接的的 DNS 地址：

  • github.com
  • assets-cdn.github.com
  • github.global.ssl.fastly.net

2 修改配置文件

打开系统 hosts 文件(需管理员权限)。 路径：C:\Windows\System32\drivers\etc,

3 刷新DNS地址

在命令行输入：ipconfg /flushdns

原文出处：https://bestzuo.cn/posts/497652030.html
