第一步 如何配置

在我们的自定义的包，添加一个python文件，且该文件必须命名为：setup.py，然后在该文件中，写上一些具体的参数即可。

这里只介绍一些基本的参数，具体的所有参数，请参考setuptools官网。

第二步 如何使用

我们有两种方式可以在本地运行，这两种方法的前提都是得进入到你的包的目录中：

  • 方法一：使用python setup.py install进行安装
  • 方法二：使用pip install .进行安装，注意：有一个点，别忘了，代表当前目录

第三步 如何联网安装

当然，如果你不满足于本地，希望将自己的包给全世界的程序员使用，那么你可以将自己的上传至PYPI，这样的话，我们就可以直接使用pip install your-package-name进行联网安装了，并不需要进入到你的目录里面。
