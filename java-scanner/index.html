2 原因解释

因为scanner.nextLine()会接收一行数据，而scanner.next()是接收有效字符（除空格和enter等之外的所有字符）之前的数据，所以当输入完age值后，按下enter键之后，这个enter被scanner.nextLine()给吸收了，也就被其当做一个值给接收了，所以就自动跳过了中间的name，所以会导致出现bug。

不过没关系，解决办法就是在需要真正接收的变量前加一个空的scanner.nextLine()，将上一个输入的enter给接收了，这样就不会出bug了。
