右键点击【我的电脑】-》【管理】-》【服务和应用程序】-》【服务】-》关闭MySQL服务

2003 - can’t connect to mysql server on ‘localhost’ (10061)
出现这种错误，就是说明mysql服务没有启动，可以通过查看服务页面：

如果现实没有启动，那么有下面几种情况可以检查：

情况2

检查MySQL的配置文件的读取路径，如何查看读取路径，进入到【服务】窗口，查看MySQL的属性，可以看到如下所示：

如果该路径下没有my.ini配置文件，那么我们有两种方法解决：

  • 将新目录下的配置文件拷贝到默认的路径下，不需要修改读取路径
  • 修改配置文件的读取路径，不需要拷贝配置文件

这里介绍如何修改读取路径，拷贝文件就不用介绍了吧。编辑新目录下的my.ini文件，修改这一部分，将路径修改成你的配置文件所在路径即可。
