3 nonlocal

在介绍nonlocal关键字之前，大家需要先了解一下闭包的概念，闭包在很多的编程语言中都有，在python中也有。

什么是闭包？

闭包就是在一个函数嵌套另一个函数，这两个函数都有其局部变量，外部的函数中的变量，在内部函数中是不能被直接修改的。也就是说，我们可以进行间接修改。那间接修改就需要用到nonlocal关键字。
